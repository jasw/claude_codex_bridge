# agentroles.ccb_round_reviewer

Draft accepted RolePack for immaculate, dynamic whole-round verification after
controller-owned node review and integration.

The round reviewer creates a round report with an explicit machine-readable
result line from compact node-review, integration, project-root, authority, and
cleanup evidence. CCB scripts import that evidence and decide authoritative
state transitions; the reviewer cannot mark the task or round done.
